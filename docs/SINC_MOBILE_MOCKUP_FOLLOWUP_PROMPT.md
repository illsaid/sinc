Design 3 additional mobile app screens for **Sinc**, matching the exact visual language of the existing mockup:
- warm off-white / bone background
- quiet editorial design
- refined serif headline typography mixed with clean sans-serif UI text
- calm premium product feel
- tactile card objects
- minimal muted accents
- no bright gradients

The existing screen already works as the **share-sheet confirmation / capture** moment.
These new screens should look like they belong to the same app and the same product system.

## Product concept
Sinc is a calm memory layer for the disappearing social web.
It helps users catch posts, videos, articles, and images before they vanish, then route them into a small number of meaningful shelves so they can actually find them again later.

The product must visually communicate:
- saving and keeping are different actions
- no giant blob of saved stuff
- finite shelves
- archive as memory, not obligation
- retrieval and re-finding are central
- the app is humane, calm, and shelf-like

## Create these 3 screens

### 1. Home screen
A calm overview screen.
Include:
- a small **New to resolve** module with 2–3 recent saved items
- a **Shelves** section with a few shelves like Film, Design, Health, Music
- each shelf showing a waiting count
- a **Recently kept** strip or row
- one resurfaced older item

Important:
- do not make it look like a giant inbox
- do not make it look like a dashboard
- everything should feel finite and manageable
- the composition should feel quiet and elegant

### 2. Shelf screen
A finite shelf view, like a playlist or curated stack.
Example shelf: **Design**
Include:
- shelf title
- visible count / progress indicator
- 4–5 saved items in a bounded list or card stack
- preview image, source, time saved
- subtle actions such as Keep, Dismiss, Wrong fit, Open

Important:
- this screen must feel processable
- not endless
- not like social scrolling
- not like a to-do app

### 3. Archive / search screen
A quiet searchable memory layer.
Include:
- search bar
- filters for shelf / platform / date
- preview-first search results
- greyed or softened archived items
- elegant way of showing older saved material

Important:
- this should feel like memory and retrieval
- not backlog, not obligation, not clutter
- calm, deliberate, and slightly editorial

## Content examples
Use believable saved content such as:
- Japanese cinema stills
- design objects / interiors
- essays / Substack articles
- music / album imagery
- cultural and aesthetic media

## Composition notes
- show the 3 screens together in one clean presentation board or carousel
